Cyber Tools We Use

1. Nmap

Used for network scanning and finding open ports on systems.

2. Wireshark

Helps to capture and analyze network traffic for better understanding of data flow.

3. SQLMap

Finds and exploits SQL injection bugs in web applications automatically.

4. Burp Suite

Used to test web application security and find vulnerabilities like XSS or SQL injection.

5. Metasploit

Used by ethical hackers for penetration testing and exploiting system vulnerabilities.
